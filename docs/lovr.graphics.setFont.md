lovr.graphics.setFont
===

Set the active font used to render text.

    lovr.graphics.setFont(font)

### Arguments

- `Font font` The new font to use.

### Returns

Nothing

---

Reset back to the default font.

    lovr.graphics.setFont()

### Arguments

None

### Returns

Nothing

See also
---

- `lovr.graphics.getFont`
- `Font`
