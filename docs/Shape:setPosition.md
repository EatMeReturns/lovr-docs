Shape:setPosition
===

Set the position of the shape, relative to the collider it's attached to.

    shape:setPosition(x, y, z)

### Arguments

- `number x` The shape's new x offset.
- `number y` The shape's new y offset.
- `number z` The shape's new z offset.

### Returns

Nothing
